# ML Prediction Project Agent

## Overview
This agent automates the creation of machine learning prediction projects that combine:
- **Data Exploration & Analysis**: Jupyter Notebook for EDA and model development
- **Model Training**: Machine learning pipeline using scikit-learn/xgboost
- **Web Interface**: Streamlit application for interactive predictions
- **Model Persistence**: Serialized ML models using joblib

## Project Structure

```
project-root/
├── app.py                          # Streamlit web application
├── requirements.txt                # Python dependencies
├── notebooks/
│   └── PRACTICE_EX.ipynb          # Data exploration & model training
├── data/
│   └── [dataset].csv               # Input data (CSV format)
└── models/
    └── [model_name].pkl            # Trained model (joblib serialized)
```

## Key Components

### 1. Data Preparation (Jupyter Notebook)
- Load and explore dataset with pandas
- Perform exploratory data analysis (EDA)
- Handle categorical variables (one-hot encoding)
- Feature engineering and preprocessing
- Train machine learning model
- Evaluate model performance
- Save trained model using joblib

**Key Libraries:**
- pandas: Data manipulation
- numpy: Numerical operations
- matplotlib/seaborn: Visualization
- scikit-learn: Model training and preprocessing
- xgboost: Gradient boosting models (optional)

### 2. Model Training Pipeline
- Load raw CSV data
- Apply feature transformations and encoding
- Split data into train/test sets
- Train regression/classification model
- Evaluate metrics (accuracy, RMSE, etc.)
- Serialize model to `.pkl` file in `models/` directory

### 3. Streamlit Application (app.py)
- Load pre-trained model from `models/` directory
- Create interactive UI with user input controls
- Accept numerical and categorical inputs
- Perform one-hot encoding matching training data
- Generate and display predictions
- Display results with formatted output

**Standard Streamlit Components:**
- st.title() - App title
- st.text_input() - Text inputs
- st.number_input() - Numeric sliders
- st.selectbox() - Dropdown selections
- st.button() - Prediction trigger
- st.success()/st.error() - Result display

## Workflow for New Projects

### Step 1: Dataset Preparation
1. Place new CSV dataset in `data/` directory
2. Document dataset structure and columns
3. Identify target variable and features

### Step 2: Data Exploration Notebook
1. Create/update `notebooks/PRACTICE_EX.ipynb`
2. Load dataset: `pd.read_csv("../data/[filename].csv")`
3. Perform EDA:
   - Display shape and info
   - Check for missing values
   - Analyze distributions and relationships
   - Visualize key patterns
4. Identify categorical vs numerical features
5. Handle missing values and outliers

### Step 3: Feature Engineering & Encoding
1. Extract relevant features
2. Apply transformations if needed:
   - Scaling: StandardScaler, MinMaxScaler
   - Encoding: One-hot encoding for categorical variables
   - Feature creation: Domain-specific calculations
3. Create preprocessor pipeline
4. Document encoding mappings for app consistency

### Step 4: Model Training
1. Split data: train_test_split()
2. Train model using appropriate algorithm:
   - Regression: LinearRegression, RandomForestRegressor, XGBRegressor
   - Classification: LogisticRegression, RandomForestClassifier, XGBClassifier
3. Evaluate with appropriate metrics
4. Save model: `joblib.dump(model, "../models/[model_name].pkl")`

### Step 5: Streamlit Application
1. Update `app.py` with:
   - Load model from correct path
   - Define input fields matching training features
   - Adjust min/max values for number_input based on data
   - Update continent_data dict (or equivalent encoding mapping) to match features
   - Build input DataFrame with exact column names and encoding format
   - Format prediction output appropriately

### Step 6: Environment & Dependencies
1. Create `requirements.txt` with needed packages:
   ```
   streamlit>=1.0
   pandas>=1.3.0
   joblib>=1.0
   scikit-learn>=1.0
   matplotlib>=3.3.0
   seaborn>=0.11.0
   numpy>=1.20.0
   ```
2. Install: `pip install -r requirements.txt`
3. Create virtual environment: `python -m venv .venv`

### Step 7: Running the Application
```bash
# Activate environment
.\.venv\Scripts\Activate.ps1  # Windows PowerShell

# Run Streamlit app
streamlit run app.py
```

## Critical Implementation Notes

### One-Hot Encoding Consistency
- **MUST match exactly** between notebook and app
- If training encodes continents as: `continent_Asia`, `continent_Europe`, etc.
- App must create DataFrame with identical column names
- Drop first category to avoid multicollinearity (e.g., Africa dropped)
- Only set one continent column to True per prediction

### Model Serialization
- Use `joblib` for pickling models: `joblib.dump(model, path)`
- Load with: `joblib.load(path)`
- Store in `models/` directory with `.pkl` extension

### Input Validation
- Set realistic min/max bounds for numeric inputs
- Use selectbox for categorical features (prevents invalid entries)
- Build input DataFrame before prediction with all required columns

### Feature Naming
- Input feature names must exactly match training feature names
- Maintain consistent naming across notebook and app
- Document all transformed feature names

## Common Adaptations

## Example Model Training Pattern

```python
# In notebook:
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
import joblib

# Load and prepare data
X = processed_features
y = target_variable

# Train model
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = RandomForestRegressor(n_estimators=100, random_state=42)
model.fit(X_train, y_train)

# Save model
joblib.dump(model, "../models/prediction_model.pkl")
```

## Example Streamlit App Pattern

```python
# In app.py:
import streamlit as st
import pandas as pd
import joblib

model = joblib.load("models/prediction_model.pkl")

st.title("Prediction App")

# Get inputs
feature1 = st.number_input("Feature 1", min_value=0, max_value=100)
feature2 = st.selectbox("Feature 2", ["Option A", "Option B"])

# Create input data with exact encoding
input_df = pd.DataFrame([{
    "feature1": feature1,
    "feature2_A": feature2 == "Option A",
    "feature2_B": feature2 == "Option B"
}])

# Predict
if st.button("Predict"):
    prediction = model.predict(input_df)
    st.success(f"Result: {prediction[0]:.2f}")
```
